# CodeAlpha Tasks

This repository contains front-end development tasks completed as part of the CodeAlpha program.  
All projects are developed using HTML, CSS, and JavaScript, focusing on functionality, clean UI, and user interaction.

---

## Task 1: Age Calculator

### Project Description
The Age Calculator is a simple web application that calculates a user's age based on their date of birth.  
It provides accurate age details in years and helps users quickly determine their age through an easy-to-use interface.

### Technologies Used
- HTML5  
- CSS3  
- JavaScript  

### Features
- User-friendly interface  
- Date of birth input  
- Accurate age calculation  
- Instant result display  
- Responsive design

## Task 2: To-Do List

### Project Description
The To-Do List application allows users to create, manage, and organize daily tasks efficiently.  
Users can add, delete, and track tasks, helping improve productivity and task management.

### Technologies Used
- HTML5  
- CSS3  
- JavaScript  

### Features
- Add new tasks  
- Delete completed or unwanted tasks  
- Simple and clean UI  
- Real-time task updates  
- Responsive layout
